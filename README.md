# Arduino
Arduino code Repository
18회 임베디드 소프트웨어 공모전 참여 주제.

아두이노 우노 보드.


# 기능 설명
- RTC(Real Time Clock) 모듈 사용
- I2C Display에 자세한 정보 출력
- 택트 스위치를 이용하여 설정 변경
- 화분 습도 센서를 활용하여 습도가 일정 % 이하로 내려가면 작동.
- 잦은 작동 방지를 위해 일정 시간 지나면 작동. (조건 추가)
- Relay module 사용하여 모터를 제어
